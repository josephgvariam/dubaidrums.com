Drum Classes

African Djembe Lessons

If you would like to learn the African Djembe, we run great group drum sessions.
It’s a great way to de-stress and have rhythmic fun in a group and at the same time learn an amazing percussion instrument.

Venue: DUCTAC, Mall of the Emirates, 2nd Floor
Date: Every Monday
Time: 7.30pm - 8.30pm
Price: Dhs 50 per session (drums provided)

 

Arabic Tabla Lessons

Learn to play the Arabic Tabla in a group setting with Dubai Drums own master tabla play Zaid!!

Venue: Urban Tribe Studio, Street 17, Al Quoz 4. Office Location Map

Date: Every Saturday during September
Time: 7.00pm - 8.00pm
Price: Dhs 50 per session (drums can be provided)
